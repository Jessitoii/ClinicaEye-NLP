# ClinicaEye-NLP Web Experience

The "Lamborghini" of medical interfaces. Built with **Next.js (App Router)** and **Tailwind CSS**.

## Philosophy
Clinical software shouldn't be ugly. This UI is designed for high-stress environments where clarity and speed are paramount.
- **Rich Aesthetics**: Vibrant, high-contrast themes for readability.
- **Zero Friction**: Efficient data entry and immediate AI feedback.
- **Glassmorphism & Micro-animations**: Enhanced depth and interaction feedback.

## Tech Stack
- **Next.js 14/15**: Utilizing Server Components for performance.
- **Shadcn UI**: For baseline accessible components.
- **Framer Motion**: For fluid transitions and micro-interactions.
- **Lucide React**: Purpose-specific medical icons.

## Setup
1. `npm install`
2. `npm run dev`
